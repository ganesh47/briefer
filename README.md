# Briefer

Briefer is an iOS app for concise lessons, quizzes, and spaced-repetition review. The public repo tracks app mechanics, architecture, and implementation work only; private learning content and source material stay outside GitHub.

## Product Shape

- Short lesson briefs drawn from approved in-app seed data.
- Quiz-first review loops with correction and explanation.
- Spaced repetition for concepts, glossary terms, and scenario questions.
- Learner progress, streaks, weak-area review, and session history.
- Content-ingestion pipeline with private-source boundaries.
