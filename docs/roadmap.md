# Briefer Roadmap

## Foundation

- Create the iOS project scaffold.
- Define core learning data models.
- Seed a minimal generic lesson and quiz set.
- Build local persistence.

## Learning Loop

- Implement today's review queue.
- Add quiz answer checking and explanations.
- Add spaced-repetition scheduling.
- Track attempts, confidence, and weak areas.

## App Experience

- Build Home, Brief, Quiz, Review Queue, and Progress screens.
- Add empty, loading, completion, and incorrect-answer states.
- Tune copy and interaction for fast repeated study.

## Content Pipeline

- Keep private source material out of GitHub surfaces.
- Import or transform approved content outside the public repo when needed.
- Generate reusable lesson, glossary, and question payloads.
- Add validation for duplicate or low-quality questions.
